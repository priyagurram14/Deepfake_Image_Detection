# Deepfake_Image_Detection

This project includes a deepfake image classifier with a backend (Python/Flask) and a frontend (React).  
The frontend was bootstrapped with Create React App.

## Running the Frontend

In the `frontend` folder, you can run:

### `npm start`
Runs the React app in development mode at:  
http://localhost:3000

### `npm run build`
Builds the frontend for production into the `build` folder.

## Running the Backend

- `se_back.py` (main API)
